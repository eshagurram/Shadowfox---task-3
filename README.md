# Shadowfox---task-3
Sakthi Dental Clinic Website
This project is a responsive and user-friendly website created for Sakthi Dental Clinic. The website is designed to provide clear information about the clinic, its dental treatments, frequently asked questions, and an easy way for patients to contact the clinic or book an appointment.

The website focuses on a clean and professional healthcare design and works smoothly across desktop, tablet, and mobile devices. It also includes a contact form with basic client-side validation and success/error messages to enhance user interaction.

Technologies Used:

HTML

CSS

JavaScript

Bootstrap

Key Features:

Fully responsive design for all devices

Clean and professional healthcare UI

Appointment/contact form with client-side validation

Easy navigation and structured layout
